h2. Installing the Application

* git clone git@github.com:dotgee/geocms.git
* cd geocms
//trust rvmrc
* gem install budler // if needed
* bundle install
* modify config/database.yml

* rake db:create
* rake db:migrate
* rake db:populate
* rails s // to start server
